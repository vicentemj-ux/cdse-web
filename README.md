# CDSE - Centro de Soluciones Electrónicas

Modern, professional electronics repair website for CDSE in Los Mochis, Sinaloa. Built with Astro and Tailwind CSS for optimal performance.

## Features

- **Zero-JS Information Sections**: Static pages for SEO and performance
- **React Islands**: Interactive repair status tracker and forms
- **Modern Tech Corporate Design**: Clean, professional interface with Electric Blue and Tech Teal accents
- **Lead Generation Focus**: Prominent WhatsApp CTA for repair quotes
- **Product Showcase**: Dynamic product grid with stock status
- **Trust Signals**: Brand partnerships, testimonials, and 15-year excellence badge
- **Responsive Design**: Mobile-first approach for all devices

## Project Structure

```
src/
├── components/        # Astro + React components
├── layouts/          # Base layout templates
├── pages/            # Astro pages (routes)
├── styles/           # Global styles and design tokens
└── public/           # Static assets
```

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Technology Stack

- **Framework**: Astro 5.5+
- **Styling**: Tailwind CSS 3.4+
- **Interactive Components**: React 19 (minimal islands)
- **Package Manager**: npm

## Design System

### Colors
- **Primary**: Electric Blue (#007BFF)
- **Secondary**: Tech Teal (#17A2B8)
- **Dark**: Navy (#0A1628)
- **Light**: Light Gray (#F8FAFC)

### Typography
- **Font Family**: Inter
- **Heading Weight**: 700-800
- **Body Weight**: 400-600
