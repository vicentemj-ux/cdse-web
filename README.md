# CDSE - Centro de Soluciones Electrónicas

Sitio web profesional y moderno para CDSE, negocio de reparación y venta de computadoras, celulares y videojuegos en Los Mochis, Sinaloa. Construido con Astro y Tailwind CSS.

## Características

- **Secciones de información sin JavaScript**: Páginas estáticas optimizadas para SEO y rendimiento
- **Islas de React**: Componentes interactivos para seguimiento de reparaciones y formularios
- **Diseño corporativo tecnológico moderno**: Interfaz limpia y profesional con tonos azul eléctrico y teal tecnológico
- **Enfoque en generación de leads**: Botón de WhatsApp destacado para cotizaciones
- **Catálogo de productos**: Cuadrícula dinámica de productos con estado de inventario
- **Señales de confianza**: Marcas partners, testimonios y distintivo de 16 años de excelencia
- **Diseño responsivo**: Enfoque mobile-first para todos los dispositivos

## Estructura del Proyecto

```
src/
├── components/        # Componentes de Astro + React
├── layouts/           # Plantillas de diseño base
├── pages/             # Páginas de Astro (rutas)
├── styles/            # Estilos globales y tokens de diseño
└── public/            # Archivos estáticos
```

## Cómo Empezar

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build

# Vista previa de la build de producción
npm run preview
```

## Stack Tecnológico

- **Framework**: Astro 5.5+
- **Estilos**: Tailwind CSS 3.4+
- **Componentes interactivos**: React 19 (islas mínimas)
- **Gestor de paquetes**: npm

## Sistema de Diseño

### Colores
- **Primario**: Azul Eléctrico (#007BFF)
- **Secundario**: Teal Tecnológico (#17A2B8)
- **Oscuro**: Azul Marino (#0A1628)
- **Claro**: Gris Claro (#F8FAFC)

### Tipografía
- **Fuente**: Inter
- **Peso de títulos**: 700-800
- **Peso de texto**: 400-600

## Listo para Integración de API

Todos los componentes basados en datos (estado de reparación, catálogo de productos, precios) están estructurados para aceptar datos JSON de APIs externas. Actualmente usando datos de demostración.
